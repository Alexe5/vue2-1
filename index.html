Задание:

Форма должна содержать следующие поля:

  • Name
  • Phone
  • Email
  • Some field 1
  • Some field 2

Для каждого поля должна быть реализована проверка на корректное заполнение. При некорректном вводе появляется иконка с подходящим изображением. Это должно быть реализовано путем добавления класса. Если поле заполнено корректно, то должна появляться иконка с подходящим изображением.

Должен быть индикатор выполнения, который реагирует на корректное заполнение вышеуказанных полей.

Корректное заполнение каждого поля добавляет 20% в индикаторе выполнения.

Изначально, когда поля еще пустые, иконки быть не должно.

Использовать в работе указанный ниже массив.

        info: [
            {
                name: 'Name',
                value: '',
                pattern: /^[a-zA-Z ]{2,30}$/
            },
            {
                name: 'Phone',
                value: '',
                pattern: /^[0-9]{7,14}$/
            },
            {
                name: 'Email',
                value: '',
                pattern: /.+/
            },
            {
                name: 'Some Field 1',
                value: '',
                pattern: /.+/
            },
            {
                name: 'Some Field 2',
                value: '',
                pattern: /.+/
            }
        ]

Ниже должна быть кнопка "Summary data", которая находится в неактивном состоянии до корректного заполнения всех полей.

После нажатия появляется второй экран, на котором отображается сводная информация по полям.
